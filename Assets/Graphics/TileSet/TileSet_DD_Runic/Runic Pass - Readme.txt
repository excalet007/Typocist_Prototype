Asset Pack Title: Runic Pass - Tileset Expansion
Category: Tileset Expansion
Graphic Style: Retro, pixel

This asset pack is best used with The Dark Dungeon tileset.
This tileset features circular columns, altars, rune-inscribed walls, glowing symbols, wooden walkway, and a pit tileset.

--

This package contains:
- 80+ tiles; 16x16 each
- 1x, 2x, and 3x scale

--

How to use:
- Extract the tileset
- Import the tileset in your game editor and make sure to set the tile dimensions to 16x16
